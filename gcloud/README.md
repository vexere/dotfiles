# Introduction to GCloud alias
We chose gg as alias prefix (google), not using g because git is more popular so git deserve the sorted alias g

This alias is best used with fzf installed so you can enjoy fuzzy selection features across google cloud projects, clusters, compute instances...

# GCloud alias table
| Alias | Explanation |
| -- | -- |
| `ggsa` | GooGle Select Account |
| `ggac` | GooGle Activate Configuration |
| `ggsp` | GooGle Select Project |
| `ggcp` | GooGle Current Project |
| `ggsc` | GooGle Select (or Switch) Cluster |
| `ggsnd` | GooGle Select Node and Describe |
| `ggsn` | GooGle Select Node |
| `ggsqld` | GooGle cloud SQL select and Describe |
| `ggsvc` | GooGle get all SerViCes |
| `ggssh` | GooGle select compute instance and SSH |
| `ggcc` | GooGle open Cloud Console web |
